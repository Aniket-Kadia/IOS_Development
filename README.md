# IOS_Development

**This demo project is built on the following topics:**
- UIKit
- Auto Layout
- UI Components
- UI Navigation Controller
- Web Service
- CocoaPods
- JSON Parsing and Codable protocol with URL session and Alamofire
- MVVM Architecture
